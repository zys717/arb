## 文档说明

### 目的

本文档整理了中国和美国关于民用无人机（UAV/UAS）飞行的核心法规条款，为AirSim-RuleBench项目覆盖范围

- **中国**:

  - 《无人驾驶航空器飞行管理暂行条例》（2024年1月1日生效）
  - 《民用无人驾驶航空器运行安全管理规则》（CCAR-92部，2024年1月1日生效）
- **美国**:

  - 14 CFR Part 107 - Small Unmanned Aircraft Systems（小型无人机系统规则）
  - FAA相关咨询通告和指导文件

### 无人机分类标准

#### 中国分类（按重量和性能）

| 类型           | 定义                                   | 关键参数                                |
| -------------- | -------------------------------------- | --------------------------------------- |
| **微型** | 空机重量 < 0.25kg                      | 最大飞行真高 ≤ 50m，最大速度 ≤ 40km/h |
| **轻型** | 空机重量 ≤ 4kg，最大起飞重量 ≤ 7kg   | 最大速度 ≤ 100km/h，需空域保持能力     |
| **小型** | 空机重量 ≤ 15kg，最大起飞重量 ≤ 25kg | 需空域保持能力和可靠监视能力            |
| **中型** | 最大起飞重量 ≤ 150kg                  | 不包括微型、轻型、小型                  |
| **大型** | 最大起飞重量 > 150kg                   | 需适航许可                              |

**法规来源**: 《无人驾驶航空器飞行管理暂行条例》第六十二条

#### 美国分类（按重量）

| 类型                | 定义                      | Part 107适用性          |
| ------------------- | ------------------------- | ----------------------- |
| **Small UAS** | 起飞重量 < 55磅（约25kg） | 适用Part 107            |
| **Large UAS** | 起飞重量 ≥ 55磅          | 适用Part 91，需特殊批准 |

**法规来源**: 14 CFR § 107.3

## 中国法规体系

### 1. 空域分类与限制

#### 1.1 管制空域（Controlled Airspace）

**定义**: 需经批准才能飞行的空域

**法规依据**: 《条例》第十九条

**包括区域**:

1. **高度限制**

   - 真高120米以上空域
   - **测试场景**: S006 - 统一高度上限检测
2. **地理限制区域**

   - 机场及周边一定范围 → **S004 机场禁飞区**
   - 国界线、边境线向我方一侧一定范围
   - 军事禁区、军事管理区、监管场所等涉密单位及周边 → **S001/S003 军事基地禁飞**
   - 核设施控制区域、易燃易爆危险品仓储区域 → **S002 危险区域禁飞**
   - 重要革命纪念地、重要不可移动文物及周边 → **文化遗产保护区**
3. **重要基础设施**

   - 发电厂、变电站、加油站、供水厂
   - 公共交通枢纽、航电枢纽、重大水利设施
   - 港口、高速公路、铁路电气化线路及周边
   - **测试场景**: S005 - 基础设施禁飞区
4. **特殊保护设施**

   - 射电天文台、卫星测控站、航空导航台、雷达站及周边
   - 需要电磁环境特殊保护的设施
   - **测试场景**: 电磁敏感区域限制

**条例原文**:

```
第十九条第二款：真高120米以上空域，空中禁区、空中限制区以及周边空域，
军用航空超低空飞行空域，以及下列区域上方的空域应当划设为管制空域：
（一）机场以及周边一定范围的区域；
（二）国界线、实际控制线、边境线向我方一侧一定范围的区域；
（三）军事禁区、军事管理区、监管场所等涉密单位以及周边一定范围的区域；
...
```

#### 1.2 适飞空域（Uncontrolled Airspace）

**定义**: 管制空域范围以外的空域，微型、轻型、小型无人机可自由飞行

**法规依据**: 《条例》第十九条第四款

**限制条件**:

- 高度 ≤ 120米真高
- 远离管制空域边界
- 遵守其他运行规则（速度、视距等）

**测试场景**: S019 - 适飞空域免审批运行

#### 1.3 临时管制空域（Temporary Flight Restrictions）

**定义**: 因特殊活动临时增加的管制空域

**法规依据**: 《条例》第二十条

**通知时限**:

- 重大活动: 生效前24小时发布公告
- 紧急任务（反恐、救灾等）: 生效前30分钟发布

**测试场景**: S005 - 临时禁飞区（TFR）动态识别

### 2. 飞行高度限制

#### 2.1 基础高度限制

**规则**: 管制空域从真高120米开始

**法规依据**: 《条例》第十九条第二款

**例外情况**:

- 特定类型无人机在特定条件下可申请更高高度
- 需空中交通管理机构批准

**测试场景**:

- S006: 120米高度上限检测
- S007: 分区高度限制（城市vs郊区）
- S008: 结构物附近高度豁免

#### 2.2 融合飞行高度要求

**规则**: 与有人驾驶航空器同空域飞行时的高度限制

**法规依据**: 《条例》第二十二条

**允许融合飞行的情况**:

- 大型无人机（取得适航许可）
- 中型无人机不超过真高300米
- 小型无人机不超过真高300米
- 轻型无人机在适飞空域上方不超过真高300米

**条例原文**:

```
第二十二条第二款：属于下列情形之一的，经空中交通管理机构批准，
可以进行融合飞行：
（一）根据任务或者飞行课目需要...
（二）取得适航许可的大型无人驾驶航空器的飞行；
（三）取得适航许可的中型无人驾驶航空器不超过真高300米的飞行；
（四）小型无人驾驶航空器不超过真高300米的飞行；
（五）轻型无人驾驶航空器在适飞空域上方不超过真高300米的飞行。
```

### 3. 速度限制

#### 3.1 轻型无人机速度限制

**规则**: 最大平飞速度不超过100千米/小时

**法规依据**: 《条例》第六十二条第三款

**测试场景**: S009 - 全局速度上限检测

#### 3.2 微型无人机速度限制

**规则**: 最大平飞速度不超过40千米/小时

**法规依据**: 《条例》第六十二条第二款

#### 3.3 特定区域速度限制

**规则**: 在适飞空域内，小型无人机应遵守限速规定

**法规依据**: 《条例》第三十二条第六款

**条例原文**:

```
第三十二条（六）操控小型无人驾驶航空器在适飞空域内飞行的，
应当遵守国家空中交通管理领导机构关于限速、通信、导航等方面的规定；
```

**测试场景**: S010 - 居民区低速飞行限制

---

### 4. 时间限制

#### 4.1 夜间飞行要求

**规则**: 夜间或低能见度条件下飞行需开启灯光系统

**法规依据**: 《条例》第三十二条第七款

**条例原文**:

```
第三十二条（七）在夜间或者低能见度气象条件下飞行的，
应当开启灯光系统并确保其处于良好工作状态；
```

**测试场景**: S011 - 夜间飞行灯光要求

#### 4.2 特定时段禁飞

**规则**: 某些区域（如医院）可能有时间限制

**法规依据**: 地方性规定或临时管制

**测试场景**: S012 - 时间窗口限制（如医院夜间禁飞）

---

### 5. 视距与控制要求

#### 5.1 视距内飞行要求

**规则**: 微型无人机必须保持视距内飞行

**法规依据**: 《条例》第三十二条第五款

**条例原文**:

```
第三十二条（五）操控微型无人驾驶航空器的，应当保持视距内飞行；
```

**测试场景**: S013 - 视距内飞行（VLOS）要求

#### 5.2 超视距飞行

**规则**: 超视距飞行需采取特殊措施

**法规依据**: 《条例》第三十二条第八款

**条例原文**:

```
第三十二条（八）实施超视距飞行的，应当掌握飞行空域内其他航空器的
飞行动态，采取避免相撞的措施；
```

**测试场景**: S014 - 超视距飞行（BVLOS）审批需求

---

### 6. 避让规则

#### 6.1 避让优先级

**规则**: 无人机必须避让其他航空器和地面交通工具

**法规依据**: 《条例》第三十三条

**避让顺序**:

1. 避让有人驾驶航空器 → **最高优先级**
2. 避让无动力装置的航空器
3. 避让地面、水上交通工具
4. 单架飞行避让集群飞行
5. 微型无人机避让其他无人机

**条例原文**:

```
第三十三条 操控无人驾驶航空器实施飞行活动，应当遵守下列避让规则：
（一）避让有人驾驶航空器、无动力装置的航空器以及地面、水上交通工具；
（二）单架飞行避让集群飞行；
（三）微型无人驾驶航空器避让其他无人驾驶航空器；
（四）国家空中交通管理领导机构规定的其他避让规则。
```

**测试场景**:

- S015: 避让有人驾驶航空器
- S016: 多无人机优先级判断

---

### 7. 特殊操作限制

#### 7.1 投放物品

**规则**: 投放物品需特殊申请（农用无人机作业除外）

**法规依据**: 《条例》第三十一条第二款第二项

**条例原文**:

```
第三十一条第二款：前款规定的飞行活动存在下列情形之一的，应当依照
本条例第二十六条的规定提出飞行活动申请：
...
（二）运载危险品或者投放物品（常规农用无人驾驶航空器作业飞行活动除外）；
```

**测试场景**: S017 - 投放物品限制检测

#### 7.2 集群飞行

**规则**: 集群飞行需特殊申请

**法规依据**: 《条例》第三十一条第二款第五项、第六十二条第十二款

**集群定义**:

```
第六十二条（十二）集群，是指采用具备多台无人驾驶航空器操控能力的
同一系统或者平台，为了处理同一任务，以各无人驾驶航空器操控数据
互联协同处理为特征，在同一时间内并行操控多台无人驾驶航空器以
相对物理集中的方式进行飞行的无人驾驶航空器运行模式。
```

**测试场景**: S018 - 集群飞行规则

#### 7.3 禁止行为

**法规依据**: 《条例》第三十四条

**明确禁止**:

1. 违法拍摄军事、涉密场所
2. 扰乱机关、企事业单位或公共场所秩序
3. 妨碍国家机关工作人员执行职务
4. 投放违法宣传品或物品
5. 危及公共设施、财产安全
6. 危及他人生命健康，非法采集信息
7. 非法获取、泄露国家秘密
8. 其他法律法规禁止的行为

### 8. 审批流程

#### 8.1 需要审批的飞行

**规则**: 管制空域飞行需提前申请

**法规依据**: 《条例》第二十六条

**申请时限**:

- 常规飞行: 拟飞行前1日12时前提出申请
- 批复时限: 飞行前1日21时前作出决定
- 紧急任务: 起飞前30分钟申请，起飞前10分钟批复

**长期飞行**: 固定空域常态飞行可申请长期批准，每次飞行前1日12时前备案

**条例原文**:

```
第二十六条 除本条例第三十一条另有规定外，组织无人驾驶航空器飞行活动的
单位或者个人应当在拟飞行前1日12时前向空中交通管理机构提出飞行活动申请。
空中交通管理机构应当在飞行前1日21时前作出批准或者不予批准的决定。
```

**测试场景**: S020 - 管制空域审批流程

#### 8.2 无需审批的飞行

**法规依据**: 《条例》第三十一条

**包括情况**:

1. 微型、轻型、小型无人机在适飞空域内飞行
2. 常规农用无人机作业飞行
3. 警察、海关、应急部门在驻地不超过真高120米的飞行（需确认）
4. 民用无人机在民用机场管制地带内执行巡检等任务（需备案和确认）

**但以下情况仍需申请**:

- 通过通信基站或互联网中继飞行
- 运载危险品或投放物品
- 飞越集会人群上空
- 在移动交通工具上操控
- 分布式操作或集群飞行

**测试场景**: S019 - 适飞空域免审批判断

## 美国法规体系

### 1. 空域分类与限制

#### 1.1 空域分类（FAA Airspace Classes）

**法规依据**: 14 CFR § 107.41, § 107.43, § 107.45

| 空域类别          | 描述                       | Part 107要求                      |
| ----------------- | -------------------------- | --------------------------------- |
| **Class B** | 繁忙机场区域（如JFK、LAX） | 需ATC授权（LAANC）                |
| **Class C** | 中等流量机场               | 需ATC授权（LAANC）                |
| **Class D** | 塔台控制机场               | 需ATC授权（LAANC）                |
| **Class E** | 控制空域（地面到某高度）   | 地面起飞需授权，空中≤400ft免授权 |
| **Class G** | 非控制空域                 | 无需授权，遵守Part 107规则        |

**法规原文**:

```
§ 107.41 Operation in certain airspace.
No person may operate a small unmanned aircraft in Class B, Class C, or Class D 
airspace or within the lateral boundaries of the surface area of Class E airspace 
designated for an airport unless that person has prior authorization from 
Air Traffic Control (ATC).
```

**测试场景**:

- S001-S005: 各类禁飞区检测
- Class B/C/D对应机场管制区

#### 1.2 禁飞区（No-Fly Zones）

**类型**:

1. **机场周边** - 5英里半径内需授权
2. **国家安全区域** - 华盛顿特区、军事基地
3. **国家公园** - 禁止起降（但可飞越）
4. **临时飞行限制（TFR）** - 总统行程、体育赛事等

**工具**: B4UFLY App和LAANC系统

**测试场景**: S004 - 机场禁飞区，S005 - 临时禁飞区

---

### 2. 飞行高度限制

#### 2.1 基本高度规则

**规则**: 最大高度400英尺AGL（约122米）

**法规依据**: 14 CFR § 107.51(b)

**法规原文**:

```
§ 107.51 Operating limitations for small unmanned aircraft.
(b) The altitude of the small unmanned aircraft cannot be higher than 400 feet 
above ground level, unless the small unmanned aircraft is flown within a 400-foot 
radius of a structure and does not fly higher than 400 feet above the structure's 
immediate uppermost limit.
```

**例外**: 在建筑物400英尺半径内，可以飞到建筑物顶部上方400英尺

**测试场景**:

- S006: 400英尺（122米）高度上限
- S008: 建筑物附近高度豁免

#### 2.2 结构物附近飞行

**规则**: 可在建筑物400英尺半径内飞行，高度不超过建筑物顶部+400英尺

**目的**: 允许建筑检查、屋顶勘察等作业

**测试场景**: S008 - 结构物附近高度豁免计算

---

### 3. 速度限制

#### 3.1 最大速度

**规则**: 最大地速87节（约100英里/小时，约161公里/小时）

**法规依据**: 14 CFR § 107.51(c)

**法规原文**:

```
§ 107.51 Operating limitations for small unmanned aircraft.
(c) The groundspeed of the small unmanned aircraft may not exceed 87 knots 
(100 miles per hour).
```

**测试场景**: S009 - 全局速度上限（87节）

**注意**: 与中国轻型无人机100km/h相比，美国限制更宽松（161km/h vs 100km/h）

---

### 4. 时间限制

#### 4.1 日间飞行规则

**规则**: 白天或民用晨昏蒙影时段飞行

**法规依据**: 14 CFR § 107.29

**定义**:

- 民用晨昏蒙影（Civil Twilight）: 日出前30分钟至日落后30分钟
- 当地时间为准

#### 4.2 夜间飞行要求（2021年修订）

**规则**: 夜间飞行需满足:

1. 无人机配备防撞灯，3英里外可见
2. 操作员完成夜间飞行训练

**法规依据**: 14 CFR § 107.29（修订）

**法规原文**:

```
§ 107.29 Operation at night.
(a) No person may operate a small unmanned aircraft system during night unless:
    (1) The remote pilot in command has completed an updated initial knowledge 
        test or recurrent training...
    (2) The small unmanned aircraft has lighted anti-collision lighting visible 
        for at least 3 statute miles that has a flash rate sufficient to avoid 
        a collision...
(b) Civil twilight refers to the time period commencing 30 minutes before 
    official sunrise to official sunset (evening civil twilight).
```

**测试场景**: S011 - 夜间飞行灯光和培训要求

---

### 5. 视距与控制要求

#### 5.1 视距内飞行（VLOS）

**规则**: 必须保持视距内飞行，除非获得豁免

**法规依据**: 14 CFR § 107.31

**法规原文**:

```
§ 107.31 Visual line of sight aircraft operation.
(a) With vision that is unaided by any device other than corrective lenses, 
    the remote pilot in command, the person manipulating the flight controls 
    of the small unmanned aircraft system, and the visual observer (if one 
    is used) must be able to see the unmanned aircraft throughout the entire 
    flight in order to:
    (1) Know the unmanned aircraft's location;
    (2) Determine the unmanned aircraft's attitude, altitude, and direction 
        of flight;
    (3) Observe the airspace for other air traffic or hazards; and
    (4) Determine that the unmanned aircraft does not endanger the life or 
        property of another.
```

**允许工具**: 矫正眼镜（但不能用望远镜、FPV等）

**测试场景**: S013 - 视距内飞行要求

#### 5.2 视觉观察员（Visual Observer）

**规则**: 可使用视觉观察员辅助飞行，但需满足:

- 观察员与操作员直接沟通
- 观察员能持续看到无人机
- 观察员协助避让其他航空器

**法规依据**: 14 CFR § 107.33

**测试场景**: 视觉观察员协作场景

#### 5.3 超视距飞行（BVLOS）

**规则**: 需申请Part 107豁免

**条件**: 证明能提供等效安全水平

**测试场景**: S014 - 超视距飞行豁免需求

### 6. 避让规则

#### 6.1 主动避让

**规则**: 无人机必须主动避让所有有人驾驶航空器

**法规依据**: 14 CFR § 107.37(a)

**法规原文**:

```
§ 107.37 Operation near aircraft; right-of-way rules.
(a) Each small unmanned aircraft must yield the right of way to all aircraft, 
    airborne vehicles, and launch and reentry vehicles. Yielding the right of 
    way means that the small unmanned aircraft must give way to the aircraft 
    or vehicle and may not pass over, under, or ahead of it unless well clear.
```

**测试场景**: S015 - 避让有人驾驶航空器

#### 6.2 不得干扰紧急响应

**规则**: 不得干扰紧急救援、执法行动

**法规依据**: Part 107通用规定

**测试场景**: 紧急响应区域避让

### 7. 人员上方飞行限制

#### 7.1 人员上方飞行分类（2021年新规）

**法规依据**: 14 CFR § 107.110 - § 107.145

**分类**:

| 类别                 | 重量限制           | 人员保护要求       | 测试要求              |
| -------------------- | ------------------ | ------------------ | --------------------- |
| **Category 1** | ≤ 0.55磅（250克） | 无旋转部件外露     | 无                    |
| **Category 2** | > 0.55磅           | 通过撞击伤害测试   | 需FAA认可的符合性声明 |
| **Category 3** | > 0.55磅           | 通过更严格撞击测试 | 需FAA认可的符合性声明 |
| **Category 4** | 任意               | 需适航证           | 需Part 21适航认证     |

**法规原文**:

```
§ 107.110 Category 1 operations.
To conduct Category 1 operations:
(a) A remote pilot in command must use a small unmanned aircraft that weighs 
    0.55 pounds or less on takeoff, including everything that is on board or 
    otherwise attached to the aircraft; and
(b) No exposed rotating parts that would cause lacerations.
```

**测试场景**: 人员上方飞行分类判断

#### 7.2 移动车辆上方飞行

**规则**: Category 1可飞越移动车辆，其他需满足特定条件

**法规依据**: 14 CFR § 107.145

### 8. 操作限制

#### 8.1 禁止的操作

**法规依据**: 14 CFR § 107各条款

1. **酒精/药物影响下飞行** (§ 107.27) - 8小时内饮酒、血液酒精≥0.04%
2. **从移动车辆操作** (§ 107.25) - 除非在人口稀少区域
3. **运输危险品** (§ 107.36) - 符合DOT 49 CFR定义的危险品
4. **鲁莽或疏忽操作** (§ 107.23) - 危及生命或财产
5. **多架同时操作** (§ 107.35) - 一名操作员一次只能操作一架无人机

**法规原文示例**:

```
§ 107.27 Alcohol or drugs.
A person manipulating the flight controls of a small unmanned aircraft system 
or acting as a remote pilot in command or visual observer must comply with the 
provisions of §§ 91.17 and 91.19 of this chapter.

§ 107.23 Hazardous operation.
No person may:
(a) Operate a small unmanned aircraft system in a careless or reckless manner 
    so as to endanger the life or property of another; or
(b) Allow an object to be dropped from a small unmanned aircraft in a manner 
    that creates an undue hazard to persons or property.
```

**测试场景**:

- S017: 投放物品限制
- S018: 单操作员单无人机限制

#### 8.2 天气最低标准

**规则**: 最小能见度3英里

**法规依据**: 14 CFR § 107.51(d)

**法规原文**:

```
§ 107.51 Operating limitations for small unmanned aircraft.
(d) The minimum flight visibility, as observed from the location of the control 
    station must be no less than 3 statute miles.
```

**测试场景**: S012 - 能见度要求检测

#### 8.3 云层距离要求

**规则**: 距离云层至少500英尺

**法规依据**: 14 CFR § 107.51(e)

**法规原文**:

```
§ 107.51 Operating limitations for small unmanned aircraft.
(e) The minimum distance of the small unmanned aircraft from clouds must be 
    no less than 500 feet below the cloud and 2,000 feet horizontally from 
    the cloud.
```

## 规则维度对比

### 1. 空域管理对比

| 维度                 | 中国                   | 美国                | 测试场景 |
| -------------------- | ---------------------- | ------------------- | -------- |
| **高度限制**   | 120米真高              | 400英尺AGL（122米） | S006     |
| **机场管制**   | 机场及周边划为管制空域 | 5英里半径需ATC授权  | S004     |
| **审批机制**   | 提前1日申请，1日回复   | LAANC实时授权       | S020     |
| **免审批区域** | 适飞空域（120米以下）  | Class G空域         | S019     |

**相似性**: 两国都约定了120米左右的基础高度限制，都有机场管制区概念

**差异性**:

- 中国更强调"管制空域vs适飞空域"二分法
- 美国有更精细的空域分类（A-G类）
- 中国审批流程更正式（提前1日），美国LAANC系统更灵活

### 2. 速度限制对比

| 类型                | 中国                         | 美国            | 差异               |
| ------------------- | ---------------------------- | --------------- | ------------------ |
| **小型/通用** | 特定区域有限速（未明确全局） | 87节（161km/h） | 美国限制更明确     |
| **轻型**      | 100km/h                      | -               | 中国对轻型单独规定 |
| **微型**      | 40km/h                       | -               | 中国对微型单独规定 |

- 中国按重量分类设置不同速度上限

## 场景设计原则

### 1. 法规可追溯性

**要求**: 每个场景都应该能够追溯到具体的法规条款

**示例**:

```yaml
scenario:
  id: S001
  name: "单一圆形禁飞区检测"
  regulation_reference:
    china:
      - document: "无人驾驶航空器飞行管理暂行条例"
        article: "第十九条第二款"
        content: "军事禁区、军事管理区...应当划设为管制空域"
    usa:
      - document: "14 CFR Part 107"
        section: "§ 107.41"
        content: "No person may operate...in Class B, C, or D airspace..."
```

### 2. 跨法规对比

**要求**: 场景应该能够测试LLM对不同法规体系的理解

**示例**:

- S006场景可以测试LLM是否理解中国的"120米"和美国的"400英尺"实质相同
- S016场景可以测试LLM是否理解中国特有的多无人机优先级规则

### 3. 复杂度递进

**层级设计**:

**Level 1 - 单一规则检测**:

- S001: 单一geofence
- S006: 单一高度限制
- S009: 单一速度限制

**Level 2 - 多规则组合**:

- S002: 多个geofence
- S007: 分区高度限制
- S010: 分区速度限制

**Level 3 - 冲突解决**:

- S003: 优先级冲突
- S016: 多无人机优先级
- 组合场景: 高度+速度+geofence

### 4. 测试用例设计模板

**标准格式**:

